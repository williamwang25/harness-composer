# Figure Assets

This directory contains paper figures copied from the EMNLP LaTeX draft and converted into
GitHub-friendly PNG previews for the project README. PNG previews are exported at 600 DPI.

| File | Description |
| --- | --- |
| `overview.pdf` / `overview.png` | Harness Composer architecture: profiler, router, module library, composer, runtime loop, and manifest. |
| `terminal-bench-results.pdf` / `terminal-bench-results.png` | Terminal-Bench 2 top-line results and Mini-SWE-Agent baseline comparison. |
| `ablation-efficiency.pdf` / `ablation-efficiency.png` | Ablation and cost-efficiency analysis across Terminal-Bench 2 and SWE-bench Verified-100. |

The PDF files are the source-quality paper figures. The PNG files are used for GitHub rendering.
